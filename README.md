# Live Visitor Counter - React + Vite

An animated live visitor counter with spring physics, growing avatar stack, and smooth digit transitions.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Features

- Spring physics-based digit animations
- Dynamic avatar stack that grows/shrinks with visitor count
- Pulsing live indicator
- Pop-in animations for new avatars
- Smooth transitions and hover effects
- Random visitor count updates every 1.66 seconds
- Count ranges from 105 to 140 visitors
